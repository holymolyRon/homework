# 네 번째 회고

이번 주도 모두 출석을 무사히 완료했다! 수요일에는 전날 밤 알람을 맞추지 않고 잠이 들었는데, 몸이 스스로 깨어나서 스스로도 꽤 신기했다.

수업 중간중간 잠깐 졸기도 했지만, 잠을 깨기 위해 나름대로 노력했다. 수면 양말을 벗어보기도 하고, 앉아 있는 자세를 고쳐보거나 쉬는 시간마다 잠깐 밖에 나가 바람을 쐬기도 했다. 몸은 조금씩 익숙해지고 있는 것 같지만, 수업이 끝난 뒤 찾아오는 게으름은 아직 더 고쳐야겠다고 느꼈다.

어느덧 4주 차가 마무리되었다는 사실이 신기하다. 매일 수업에 참여하는 일상에는 익숙해졌지만, 여전히 스스로의 노력이 부족하다고 느낀다. 저번 주보다 이번 주에는 더 노력하고, 더 성장하겠다는 태도를 목표로 다시 한번 회고를 적으며 다짐해본다.
